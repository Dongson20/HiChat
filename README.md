# HiChat
实时聊天Web应用程序

<h2>这一个web实时聊天室，支持统计在线人数 支持发送 文字+图片+表情 ，主要利用NodeJs的webSocket<h2>
<p>下载或者克隆之后</p><br>
<p>根目录运行指令： node app.js </p><br>
<p>浏览器输入 localhost:3001 即可预览</p>
